# 方法一 JSON方法

```js
let a = {
    b: 1,
    c: '12',
    d:{aa:1,bb:false},
    e:[1,2,3,4,5],
}


let b = JSON.parse(JSON.stringify(a))

console.log(a)
console.log(b)
console.log(a === b)
```

方法的缺点: 

* 不支持函数
* 不支持JSON不支持的所有类型 比如undefined
* 不支持环状结构
* 不支持Date
* 不支持正则表达式
